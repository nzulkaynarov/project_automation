# Dry Fruits Telegram Bot 🍇

Telegram-бот для компании **DRY-FRUITS INDUSTRY** (Узбекистан, Самарканд), предназначенный для автоматизации мониторинга новых товаров на сайте [dry-fruits.uz](https://dry-fruits.uz/) и предоставления удобного каталога для пользователей.

## Основной функционал:
1. **Автоматический мониторинг**: Каждые 10 минут бот сканирует сайт на предмет появления новых товаров. При их обнаружении бот автоматически загружает их в локальную базу данных и делает рассылку всем пользователям, нажавшим `/start`.
2. **Интерактивный каталог**: Кнопка "📦 Все товары" выводит список всех сухофруктов, орехов и свежих фруктов в виде красивых карточек (фото + описание). Карточки можно перелистывать с помощью inline-кнопок (Вперед/Назад).
3. **Умный поиск (FSM)**: Кнопка "🔍 Поиск товара" позволяет найти нужный товар (например, "Дыня" или "Орех") с сохранением функции перелистывания (пагинации) результатов поиска.

## Технологический стек:
* **Python 3**
* **aiogram 3.x**: Асинхронный фреймворк для Telegram API.
* **BeautifulSoup4 & aiohttp**: Парсинг каталога товаров.
* **APScheduler**: Выполнение фоновых задач по расписанию.
* **SQLite**: Локальная легковесная база данных для хранения продуктов и ID пользователей.

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/nzulkaynarov/project_automation.git
   cd project_automation
   ```

2. Настройте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. Настройте переменные окружения:
   Переименуйте `.env.example` в `.env` и вставьте ваш Telegram-токен, полученный у @BotFather:
   ```env
   BOT_TOKEN=ваш_токен_здесь
   ```

4. Первичная загрузка товаров:
   Чтобы бот сразу имел каталог (а не ждал первого фонового сканирования), запустите скрипт импорта данных:
   ```bash
   python load_detailed_products.py
   ```

5. Запуск бота:
   ```bash
   python main.py
   ```

## Структура проекта:
* `main.py` — Точка входа, запуск бота и планировщика.
* `bot/handlers.py` — Обработчики сообщений и нажатий кнопок.
* `bot/database.py` — Логика взаимодействия с SQLite (сохранение юзеров, товаров).
* `bot/parser.py` — Парсер сайта.
* `bot/keyboards.py` — Reply и Inline клавиатуры.
* `bot/scheduler.py` — Фоновая проверка новых товаров каждые 10 мин.
